# Late Night Hacker HQ

A motivational hype machine for late-night coders. Smash the hype button, rack up streaks, and roll quirky project ideas — all wrapped in a glowy, terminal-aesthetic UI.

## Features

- Typing-animated motivational quotes
- Hype button with streak combos and glitch effects
- Randomized quirky project idea generator
- Floating particle background with scanline overlay
- Time-of-day aware greeting

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Built with Vite + React. Hosted on GitHub Pages.
